Nest Guard – Babysitter Management Web Application
Nest Guard is a fully designed babysitter–parent communication platform created to provide a smooth, secure, and easy childcare management experience.
The application allows parents to connect with babysitters, monitor updates, communicate instantly, and access essential features through a clean and modern interface.
The project includes both frontend development and backend integration using Node.js and MySQL.

✨Overview
The application begins with a dedicated start page, followed by a complete authentication flow that includes a login page, registration page, and a forgot password page.
Once authenticated, users can access the main home interface where navigation is simple, visually appealing, and optimized for mobile dimensions.
The home page represents the brand design and offers seamless navigation to the core sections of the application: Nannies, Features, and Profile.

📱App Pages
-The start page welcomes users with a soft-themed UI and transitions into the login and registration system.
-The login page allows users to sign in using their credentials, while the registration page captures parent information, child details, and an optional profile photo.
-The forgot password page assists users in recovering access by resetting their credentials.
-The home page showcases a clean layout with brand elements and bottom navigation.
It functions as the center hub from where parents can move to all other sections of the system.
-The Nannies page displays a structured list of babysitters, each presented with key details such as experience, pricing, and availability.
Users can sort the babysitters from low to high or high to low based on cost. A selection option enables parents to choose a suitable babysitter.
-The Features page highlights the core functionalities of the application.
All primary communication takes place through WhatsApp, making the system simple, instant, and familiar for parents and babysitters.
The app supports live chatting through WhatsApp messages, sharing of photos and short videos, routine updates for sleep, feeding, and playtime, as well as emergency alerts communicated directly by the babysitter.
Parents can also initiate WhatsApp voice calls and video calls for real-time interaction. All notifications and updates are received instantly through the same WhatsApp channel.
-The Profile page allows parents to manage their personal information, upload or modify their photo, and store important child-related data.
This page creates a centralized space for user identity inside the application.

⭐Features
Nest Guard focuses on seamless communication between parents and babysitters.
All primary child-related updates and interactions are executed through WhatsApp. 
This includes text messaging, photo and video sharing, routine tracking messages, emergency alerts, and voice or video calling. By integrating WhatsApp-based communication, the system provides a real-time, reliable, and user-friendly method for parents to stay connected with the babysitter at all times.

📁Project Structure
The project is arranged into multiple pages including the start screen, authentication pages, the home interface, the nannies listing, the features section, and the profile settings page.
The backend folder contains the Node.js server and MySQL database structure.
🎨 UI Design
-Cloud + pastel theme
-Rounded components
-Bottom navigation bar
-Mobile-app style 330px layout
-Smooth shadows & gradients
🧩 Backend (Node + MySQL)
-Secure database connection
-Stores parent, child & babysitter data
-API for communication & chat features
-Uses MySQL for permanent storage
-LocalStorage for temp data like UI preferences


🎯Purpose
Nest Guard was developed as an academic project demonstrating user interface design, structured page navigation, backend integration, data handling, and a functional communication model using WhatsApp as the medium.
The system provides a complete flow from user authentication to real-time engagement with babysitters.
